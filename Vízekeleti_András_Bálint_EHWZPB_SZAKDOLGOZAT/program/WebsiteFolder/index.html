Welcome to the website of my BSC Thesis! Let me introduce you to the concept!
My thesis is about a login system, that can authenticate users based on their keyboard patterns and behavior.
This means that the system can recognize the user based on how they type on the keyboard, thus eliminating the need to memorise a password.
This, however requires a bit of training for the model to create a keyboard profile of the user.
